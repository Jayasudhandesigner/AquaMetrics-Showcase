# AquaMetrics - Intelligent Water Management

AquaMetrics is a full-stack showcase project demonstrating proficiency in **SQL Database Design**, **HTML5**, **Modern CSS3**, and **Vanilla JavaScript**.

## 🚀 Key Features
- **Modern Dashboard UI**: Built with a "Glassmorphic Midnight" aesthetic.
- **Data Visualization**: Custom-built bar charts with smooth entry animations.
- **SQL Analytics**: Comprehensive schema for smart meter tracking with complex analytical queries.
- **Performance Optimized**: Zero external dependencies, utilizing vanilla web standards for maximum speed.

## 🛠️ Tech Stack
- **Frontend**: HTML5 (Semantic), CSS3 (Grid/Flexbox/Variables/Filters), JavaScript (ES6+).
- **Backend/Data**: PostgreSQL/MySQL (Schema & Analytical Queries).

## 📊 SQL Showcase
The database layer includes:
- Normalized relational schema.
- Advanced Leak Detection algorithm using CTEs.
- Automated billing calculation queries.
- High-performance indexing for time-series logs.

## 🎨 Web Excellence
- **Glassmorphism**: High-end visual effects using `backdrop-filter`.
- **Responsive Layout**: Fluid design from mobile to ultra-wide monitors.
- **Micro-animations**: Smooth state transitions and data loading effects.
